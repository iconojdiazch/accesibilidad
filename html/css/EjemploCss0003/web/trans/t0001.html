Para poder crear plantillas y layouts que nos permitan organizar nuestros contenidos, hay que considerar en primer lugar dos estrategias:

  • Elementos flotantes
  • Posicionamiento de elementos

Los elementos flotantes permiten romper el alineamiento estándar de elementos en HTM, de forma que un elemento sale del flujo natural en el que el navegador los coloca.

Es posible hacer flotar un elemento hacia la derecha o hacia la izquierda.

Cuando deseemos volver al flujo normal, predefinido por html, el primer elemento al que se le aplique debe ser limpiado. Esta limpieza puede hacerse a la derecha, a la izquierda o a ambos

lados.

La siguiente regla css img{float:left;} declara que en la página a que este estilo aplique, todas las imágenes flotarán a la izquierda.

Mientras que esta otra .limpiar{clear:both;} define una clase que devolverá al primer elemento al que aplique al flujo normal de alineación definido por html.

Aquí podemos ver un ejemplo

La hoja de estilo responsable es la siguiente:

  1. body { font-size:80%; font-family:'Lucida Grande',Verdana,sans-serif; margin:10px; background-color:#CCCCCC; }

  2. #container { padding:10px; border:1px solid #000000; background-color:#FFFFFF; }

  3. .image_float { float:left; margin:0 5px 5px 0; }

  4. h1 { font-size:150%; } h2 { font-size:140%; } h3 { font-size:120%; }

  5. p { font-size:100%; line-height:150%; } .spacer{ clear:left; }
