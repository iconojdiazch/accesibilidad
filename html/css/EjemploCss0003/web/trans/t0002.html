Por posición de un elemento entendemos sus coordenadas arriba, derecha, abajo izquierda en relación a algún otro, o bien a la propia ventana del navegador.

Html define un posicionamiento por defecto de todos los elementos de una página, el cual puede modificarse empleando la propiedad css position, junto con las que definen las coordenadas

de un elemento: top, left, bottom y right.

Css define cuatro tipos de posicionamiento:

Estático

Por defecto. El navegador calcula la posición de cada elemento y los coloca de izquierda a derecha y de arriba a abajo. Naturalmente, elementos como un párrafo o una tabla alteran esta linealidad hasta cierto punto.

Absoluto

El elemento se posiciona en relación con el más cercano, y ambos forman una unidad. Se desplazan conjuntamente, manteniendo siempre la misma relación entre sus posiciones.

Fijo

Un elemento se posiciona en relacióna la ventana del navegador, no a otro elemento html. Así, si se hace un scroll de la página, el elemento fijo puede permanecer permanentemente

visible, por ejemplo.

Relativo

Este modo indica que un elemento se posicionará en relación a dónde lo colocaría el navegador por defecto, esto es, en modo estático.
